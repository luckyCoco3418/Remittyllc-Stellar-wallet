# Remittyllc-Stellar-wallet
Stellar wallet for mobile. It has been developed in ReactNative.

## Preinstall
install https://slproweb.com/download/Win64OpenSSL-1_0_2r.exe
```npm install -g --production windows-build-tools```
```npm install -g ed25519```
```npm install -g yarn```

## Install sdk
```cd PROJECT_FOLDER```
```yarn add @pigzbe/react-native-stellar-sdk```
```react-native link react-native-randombytes```
```npm install -save react-native-qrcode```
